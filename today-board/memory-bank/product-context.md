# Product Context

## Target User

A beginner who wants to start the day by choosing one important focus and a few small tasks.

## User Need

The user does not need a full productivity system. They need a calm, simple board that answers:

- What should I focus on today?
- What are the few tasks I need to finish?
- What have I already completed?

## UX Tone

- Calm
- Minimal
- Korean copy should feel natural and encouraging
- No loud dashboard or heavy productivity pressure

## UI Notes

- The first screen should be the actual app, not a landing page.
- Keep inputs and actions obvious.
- Empty state copy should help the user start.
- Mobile layout must not overlap.
