# racing-game

`racing-game`은 Three.js와 Vite로 만든 3D 아케이드 레이싱 게임입니다. 플레이어는 사막 협곡 서킷에서 4대의 AI 경쟁 차량과 3랩 레이스를 진행합니다.

## 포함된 요소

- 3D 서킷과 사막 배경
- 플레이어 차량 1대 + AI 경쟁 차량 4대
- 순위, 랩, 속도, 니트로 HUD
- 트랙 부스트 패드
- `Shift` 니트로
- `R` 트랙 복귀

## 실행 방법

### 1. 프로젝트 폴더로 이동

```bash
cd racing-game
```

### 2. 의존성 설치

```bash
npm install
```

### 3. 개발 서버 실행

```bash
npm run dev
```

기본적으로 Vite가 로컬 개발 서버를 띄웁니다. 터미널에 표시되는 주소로 접속하면 됩니다. 일반적으로는 아래 주소를 사용합니다.

```text
http://localhost:5173
```

## 게임 시작 방법

1. 브라우저에서 개발 서버 주소를 엽니다.
2. 게임이 로드되면 카운트다운이 시작됩니다.
3. `GO`가 나오면 바로 출발할 수 있습니다.
4. 3랩을 먼저 완주하면 승리입니다.

키보드 입력이 바로 안 잡히면 게임 화면을 한 번 클릭한 뒤 다시 조작하면 됩니다.

## 조작 방법

- `W` / `ArrowUp`: 가속
- `S` / `ArrowDown`: 브레이크 / 후진
- `A` / `ArrowLeft`: 좌회전
- `D` / `ArrowRight`: 우회전
- `Shift`: 니트로 사용
- `R`: 차량을 현재 트랙 근처로 복귀

## 플레이 팁

- 아스팔트 밖으로 나가면 속도가 크게 줄어듭니다.
- 부스트 패드 구간을 타면 추가 가속을 받을 수 있습니다.
- 니트로는 직선 구간에서 쓰는 편이 가장 효율적입니다.

## 프로덕션 빌드

배포용 정적 파일을 만들려면 아래 명령을 실행합니다.

```bash
npm run build
```

빌드 결과물은 아래 폴더에 생성됩니다.

```text
dist/
```

로컬에서 빌드 결과를 확인하려면:

```bash
npm run preview
```

## 확인된 환경

- Node.js `v24.13.1`
- npm `11.8.0`

이 환경에서 `npm install`, `npm run build`까지 확인했습니다.
