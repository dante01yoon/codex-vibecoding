# Project Brief

## 프로젝트 요약

`realtime-feed`는 React, Vite, Vercel, Supabase로 만드는 한국어 실시간 방명록 앱이다. 방문자는 로그인 화면 없이 익명 세션으로 이름, 메시지, 무드, 아바타 색상, 선택 첨부를 남긴다.

## 대상 사용자

- 행사, 수업, 데모 페이지에 가볍게 인사를 남기는 방문자
- 같은 브라우저 세션에서 자신이 쓴 글과 댓글을 정리하고 싶은 작성자
- Supabase Realtime 흐름을 작은 예제로 배우는 학습자

## 핵심 범위

- 첫 화면에서 실제 방명록 작성과 피드 경험을 보여준다.
- 첫 UI 구현은 샘플 데이터 기반으로 시작한다.
- 따뜻한 포스트잇 갤러리 느낌을 유지한다.
- 이후 Supabase 익명 인증, 게시글, 댓글, 첨부, Realtime 연동으로 확장한다.

## 범위 밖

- 마케팅 랜딩 페이지
- 일반 로그인, 소셜 로그인, 비밀번호 계정
- 관리자 대시보드, 신고 큐, 검색, 알림, 해시태그
- 게시글당 여러 첨부 또는 전문 그림판
