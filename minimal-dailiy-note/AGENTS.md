# AGENTS.md

## Project

이 프로젝트는 초보자도 이해하기 쉬운 미니멀 일기 웹 앱입니다.

## Working Rules

- 작고 집중된 변경을 우선합니다.
- 사용자가 명시적으로 승인하지 않는 한 새 의존성을 추가하지 않습니다.
- 수정하기 전에 어떤 파일을 확인하거나 수정할 계획인지 설명합니다.
- 첫 구현은 단순하면서도 실제로 동작하게 만듭니다.
- 수정 후에는 변경한 파일, 검증 단계, 남은 위험을 요약합니다.

## Product Scope

- 사용자는 일기 내용을 작성할 수 있습니다.
- 사용자는 기분을 선택할 수 있습니다.
- 저장된 일기는 목록에 표시됩니다.
- 일기는 로컬 스토리지를 사용해 새로고침 후에도 유지됩니다.
- 빈 상태 화면은 차분하고 격려하는 느낌이어야 합니다.

## Frontend Style

- UI는 미니멀하고 차분하며 읽기 쉽게 유지합니다.
- 자연스러운 한국어 문구를 사용합니다.
- 장식적인 복잡함은 피합니다.
- 모바일 화면에서 텍스트가 겹치지 않도록 확인합니다.
- 안정적인 간격과 명확한 시각적 위계를 사용합니다.

## Do Not

- 기본 버전에는 UI 라이브러리를 설치하지 않습니다.
- 인증이나 백엔드 기능을 추가하지 않습니다.
- 작은 요청을 처리하기 위해 프로젝트 전체를 다시 작성하지 않습니다.
- 관련 없는 설정 파일을 수정하지 않습니다.

## Verification

- 사용 가능한 lint 또는 build 명령이 있으면 실행합니다.
- 자동화된 확인 방법이 없으면 수동 브라우저 확인 방법을 설명합니다.
- 일기 작성, 새로고침 후 유지, 빈 상태, 모바일 레이아웃을 확인합니다.
