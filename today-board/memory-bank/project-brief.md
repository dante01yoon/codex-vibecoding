# Project Brief

## App Name

오늘의 집중 보드

## One-Line Goal

Users can write one daily focus goal and manage a short task list for the day.

## First Version Scope

- Show today's date.
- Let the user write one focus goal.
- Let the user add tasks.
- Let the user mark tasks as done.
- Let the user delete tasks.
- Persist data with localStorage.
- Show a friendly empty state.

## Out of Scope

- Authentication
- Backend database
- Team sharing
- Calendar integration
- Notifications
- Deployment
- External UI libraries

## Success Criteria

- The app runs in the browser.
- The focus goal and tasks remain after refresh.
- The interface is readable on mobile and desktop.
- The implementation follows AGENTS.md.
