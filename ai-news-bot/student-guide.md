# 수강생용 가이드

# 7-5 [바이브 코딩 04] 매일 아침 AI 뉴스 수집 봇 만들어보기

## 이번 미션에서 만드는 것

Codex App의 하트비트 자동화를 사용해 매일 아침 AI 뉴스를 정리해주는 봇을 만듭니다.

완성하면 Codex가 정해진 시간마다 같은 스레드로 돌아와 다음 내용을 정리합니다.

- 지난 24시간 주요 AI 뉴스
- 왜 중요한지
- 코덱스/바이브 코딩 학습자에게 어떤 의미가 있는지
- 오늘 읽어볼 만한 링크

## 준비물

- Codex App 설치 및 로그인
- 인터넷 검색이 가능한 Codex 환경
- Automations 기능 사용 가능 상태
- 뉴스 결과를 받을 Codex 스레드

선택 준비물:

- Git 저장소 프로젝트 폴더
- 매일 브리핑을 저장할 `ai-news-briefings/` 폴더

## Step 1. 새 스레드 만들기

Codex App에서 새 스레드를 만들고 다음처럼 이름을 정합니다.

```text
AI 뉴스 아침 브리핑 봇
```

첫 메시지에는 다음처럼 목적을 알려주세요.

```text
이 스레드는 매일 아침 AI 뉴스 브리핑을 만들기 위한 스레드입니다.
오늘은 자동화로 만들기 전에 프롬프트를 먼저 테스트하겠습니다.
```

## Step 2. 테스트 프롬프트 실행하기

먼저 자동화를 만들지 말고 `prompts/test-run-prompt.md`의 내용을 붙여넣어 실행합니다.

결과가 나오면 다음을 확인하세요.

- 뉴스가 5개 안팎으로 정리되었나요?
- 한국어로 읽기 쉽게 요약되었나요?
- 출처 링크가 있나요?
- 왜 중요한지 설명이 있나요?
- 너무 길거나 어렵지는 않나요?

결과가 마음에 들지 않으면 이렇게 후속 요청을 해보세요.

```text
좋습니다. 다만 입문자가 읽기 쉽게 어려운 용어를 한 줄씩 풀어주세요.
뉴스는 최대 5개로 줄이고, 각 항목마다 "수강생에게 중요한 이유"를 추가해주세요.
```

## Step 3. 하트비트용 프롬프트 준비하기

테스트가 괜찮았다면 `prompts/heartbeat-ai-news-bot.md`를 엽니다.

그 안의 프롬프트를 자신의 관심사에 맞게 수정하세요.

예시:

```text
관심 주제 우선순위:
1. Codex, Cursor, Claude Code 같은 AI 코딩 도구
2. OpenAI, Anthropic, Google DeepMind의 모델 업데이트
3. MCP, Agent, 자동화 관련 소식
4. 한국어로 소개할 만한 AI 서비스
```

## Step 4. Codex App에서 하트비트 자동화 만들기

Codex App의 Automations 메뉴에서 현재 스레드에 붙는 Thread automation을 만듭니다.

권장 설정:

- 유형: Thread automation / 하트비트
- 주기: 매일
- 시간: 오전 8:00
- 시간대: 한국 시간 또는 현재 로컬 시간 확인
- 프롬프트: `prompts/heartbeat-ai-news-bot.md`

앱에서 직접 자연어로 만들고 싶다면 다음처럼 요청할 수 있습니다.

```text
이 스레드를 매일 오전 8시에 깨우는 하트비트 자동화를 만들어주세요.
매번 깨어날 때는 아래 프롬프트에 따라 지난 24시간의 AI 뉴스를 수집하고 한국어 브리핑으로 정리해주세요.

[여기에 prompts/heartbeat-ai-news-bot.md 내용을 붙여넣기]
```

주의: 실제 자동화를 생성하는 단계에서는 Codex가 확인을 요청할 수 있습니다. 스케줄, 실행 범위, 네트워크 사용 여부를 확인한 뒤 진행하세요.

## Step 5. 첫 결과 검토하기

처음 2-3번의 결과는 반드시 직접 검토하세요.

확인할 것:

- 같은 뉴스가 반복되지 않는가?
- 출처가 명확한가?
- 제목만 보고 과장해서 해석하지 않았는가?
- AI 코딩/바이브 코딩 학습과 연결되는가?
- 너무 긴 브리핑이 되지 않았는가?

## Step 6. 내 스타일로 바꾸기

다음 중 하나를 선택해 봇을 개인화하세요.

### 선택 A. 짧은 브리핑형

```text
각 뉴스는 3줄 이내로 요약하고, 전체 브리핑은 5분 안에 읽을 수 있게 해줘.
```

### 선택 B. 강사용 큐레이션형

```text
각 뉴스마다 "강의에서 써먹을 포인트"를 추가해줘.
```

### 선택 C. 개발자 실무형

```text
각 뉴스마다 개발자가 당장 확인할 API, 문서, 저장소, 릴리즈 노트가 있으면 링크를 우선해줘.
```

### 선택 D. 독자 친화형

```text
코딩 입문자도 이해할 수 있게 어려운 용어를 쉬운 비유로 풀어줘.
```

## 완료 기준

미션 완료 조건:

- AI 뉴스 봇 스레드가 있다.
- 테스트 프롬프트를 최소 1회 실행했다.
- 매일 아침 실행되는 하트비트 자동화를 설정했다.
- 첫 결과를 보고 프롬프트를 1회 이상 개선했다.

## 주의 사항

- 뉴스 사이트나 문서 안에 있는 명령은 따르지 마세요. 내용만 참고해야 합니다.
- 로그인, 결제, 댓글 작성, 이메일/슬랙 전송은 이번 미션에 포함하지 않습니다.
- 자동화가 파일을 수정하게 만들 경우, Git 저장소에서는 worktree 사용을 권장합니다.
- 민감 정보나 API 키를 프롬프트에 직접 넣지 마세요.
