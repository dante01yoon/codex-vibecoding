# AGENTS.md
## 프로젝트

이 프로젝트는 입문자도 이해할 수 있는 작은 웹앱입니다.
앱 이름은 "오늘의 집중 보드"입니다.

## 규칙들
- 전체적인 규칙과 로그 기록들은 `memory-bank/` 디렉토리 하위에 있는 파일들을 참조합니다.

## 작업 규칙

- 구현 계획을 제안하기 전에 `memory-bank/active-context.md`를 먼저 확인합니다.
- 한 번에 큰 변경을 하지 말고, 작고 집중된 변경을 선호합니다.
- 사용자가 명시적으로 승인하지 않으면 새 라이브러리를 설치하지 않습니다.
- 파일을 수정하기 전에 확인하거나 수정할 파일 후보를 먼저 말합니다.
- 작업 후에는 변경한 파일, 확인 방법, 남은 위험을 요약합니다.
- 현재 작업 사항들에 대해 먼저 `memory-bank/progress.md`를 확인하고 남은 작업을 생각합니다.
## 화면 규칙
- `memory-bank/product-context.md` 를 먼저 확인합니다.
- 1차 버전은 HTML, CSS, JavaScript만 사용합니다.
- 한국어 문구는 자연스럽고 친절하게 작성합니다.
- 모바일 화면에서 텍스트가 겹치지 않게 합니다.
- 저장은 브라우저 저장소인 localStorage를 사용합니다.

## 하지 말 것

- 1차 버전에 로그인, 백엔드, 배포, 알림 기능을 추가하지 않습니다.
- 관련 없는 파일을 다시 작성하지 않습니다.
- 브라우저에서 확인하는 방법을 설명하지 않고 성공했다고 말하지 않습니다.

## 작업이 완료될 때마다 수행해야할 사항들

- 작업 완료 사황들을 `memory-bank/progress.md`에 정리하고 작성합니다.


## 결정 사항들에 대한 기록들
- 애플리케이션 작업에 대한 결정 사항들은 `memory-bank/decision-log.md`를 확인합니다.
