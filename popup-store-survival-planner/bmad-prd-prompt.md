프로젝트 이름:
Pop-up Store Survival Planner

한 줄 설명:
팝업스토어 방문자가 방문 전 계획, 굿즈 예산, 현장 체크리스트, 생존 플랜 카드를 한 번에 준비할 수 있게 돕는 웹앱입니다.

목표 사용자:
- 인기 브랜드 팝업스토어에 방문하려는 사람
- 굿즈 구매 예산을 미리 정하고 싶은 사람
- 대기 시간이 길어도 현장에서 당황하지 않도록 준비하고 싶은 사람
- 방문 후 기록을 남기고 싶은 사람

MVP에 포함할 기능:
- 방문 날짜와 시간대 입력
- 예상 대기 시간 입력
- 굿즈 우선순위와 예산 입력
- 현장 체크리스트
- 오늘의 생존 플랜 카드 생성
- localStorage 기반 저장

MVP에서 제외할 기능:
- 로그인
- 결제
- 지도 API
- 실시간 재고 조회
- SNS 자동 공유
- 서버 저장

입문자 실습용 React 앱으로 구현할 예정이므로 범위를 작게 유지해주세요.
